### 🚀 Currently working on

**[STAGGI](https://github.com/dhia997/STAGGI)** — my final-year project (PFE), supervised at ISSAT Sousse.
An AI platform that helps students find the right internship through:

- 📄 CV upload & ATS scoring
- 🎯 Semantic job matching (Pinecone + embeddings)
- 🤖 RAG-based career chatbot (Mistral)
- 📝 Auto-generated technical quizzes with anti-cheat
- 🧾 PDF CV generation
